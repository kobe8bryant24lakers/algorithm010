# 算法训练营第一周总结

## 算法学习思路
1. 不要死磕；
2. 刻意训练；
3. 五遍法：
	a. 第一遍，5分钟内有解题思路就自己做，没有就看官方解题思路和高票答案，尽量理解，不行就背诵；
	b. 第二遍，自己写一遍；
	c. 第三遍，一天后；
	d. 第四遍，一周后；
	e. 第五遍，面试前一周。
4. 多学习高票答案，向全球精英学习；
5. 算法思路，寻找最近的重复单元，永远逃不出：分支、循环、递归。

## 学习内容回顾：
1. 数组：连续内存单元，内存控制器管理，时间复杂度（获取、搜索、插入、删除）：O(1)、O(n)、O(n)、O(n) ；
2. 链表：单向链表和双向链表，时间复杂度（获取、搜索、插入、删除）：O(n)、O(n)、O(1)、O(1) ；
3. 跳表：必须有序，时间复杂度（获取、搜索、插入、删除）：O(log(n))、O(log(n))、O(log(n))、O(log(n)) 。
4. 栈：先进后出（FILO），时间复杂度（获取、搜索、插入、删除）：O(n)、O(n)、O(1)、O(1) ；
4. 队列：先进先出（FIFO），时间复杂度（获取、搜索、插入、删除）：O(n)、O(n)、O(1)、O(1) ；
